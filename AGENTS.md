## 不可破坏的约束

- 不得把 `.env.local`、QQ 号、群号、OneBot Token 或 Codex 登录信息输出到聊天、日志或文档。
- QQ 聊天侧的 Codex 只能使用网页搜索、理解附加图片和图片生成/编辑；不得打开 Shell、文件修改、应用、插件、电脑控制或外部账号操作能力。
- 遇到不确定或不能按字面确定指代的网络梗、缩写、谐音、圈内称呼和专有词，回复或生图前必须先实时搜索并交叉核验；仍有多义就先追问，本轮不生成，禁止拆字后臆造形象。
- 当前群内“咕咕嘎嘎”默认指《明日方舟：终末地》的企鹅相关网络梗；仍要搜索当前视觉语境，不得按字面生成为格子、鸽子或鸭子的组合。
- 铃铃酱回复必须口语化、短句、有自然情绪起伏；正常回复禁止标题、编号、项目符号
  和“首先/其次/综上所述”式大纲总结。语气应随话题变化，禁止连续套用高浓度
  可爱模板；允许有小偏好和不确定，但不得编造真实经历。
- 不得让 NapCat 登录主 QQ，只能使用专用机器人小号。
- 不得把 OneBot WebSocket 暴露到公网；保持监听 `127.0.0.1` 并启用 Token。
- 不要手工永久修改 QQ 安装包入口。macOS 双启动只能使用 `scripts/macos/` 下的脚本，它们会瞬时注入并自动恢复原版入口。
- 不要误杀普通 QQ、NapCat 启动器或其他名为 `dist/index.js` 的 Node 服务；重启前必须核对 PID 和命令行。
- 群聊必须保持群白名单；直接回复仍要求 `@机器人`，但用户已明确授权普通群消息
  的话题参与和主动互动模式。不得扩大到非白名单群，也不得改成每条消息都回复。
- 早间情报只能搜索成都天气及 AI、终末地、绝区零、异环和鸣潮，必须优先可靠
  来源并附资讯链接；无可靠天气时潜水，不得把传闻当新闻。批斗必须先把当天缓存
  发言按群友归组，根据一位群友全天发言的整体表现直接发一条吐槽，不得按孤立
  单句断章取义，也不得使用主持或流程公告腔；无厘头罪名必须明显是玩笑且锚定其当天真实发言，不得编造真实黑料、违法行为或攻击
  身份、外貌、家庭、健康和隐私。轻量回应必须调用 QQ 消息表情接口，
  不能改成颜文字、表情包或额外文字。
- 加入新群、邀请成员或主动发送群消息属于外部操作，只能在用户明确授权具体目标后执行。
- 延年益寿投稿人和目标群必须显式配置，且分别是私聊白名单和群白名单的子集；
  不得把投稿扩散到其他好友或群。投稿图片必须先经 Codex 逐图审核，不能转发真人、
  年龄不明或未成年感、裸露、性行为及其他露骨内容。版权无法自动核验，提醒消息
  必须要求投稿人只提交有权转发的图片。`/取消延年益寿` 只能清空当晚待发送队列，
  不得删除已写入本机日期目录的归档原图。
## 修改与验证要求

- 配置只修改必要字段，不得整份重写 `.env.local`，并保持权限为 `600`。
- 修改铃铃酱人设时，同步更新 `.env.local`、`docs/PERSONA.md`、`src/persona.ts` 和
  `.env.example`；`tests/persona-sync.test.ts` 必须通过，防止各处提示词漂移。
- 修改 TypeScript 代码后依次运行：

```bash
pnpm typecheck
pnpm test
pnpm build
```

- 修改 macOS QQ/NapCat 启动逻辑后还要运行：

```bash
pnpm qq:verify-macos
```

- `pnpm start` 运行的是 `dist/index.js`；macOS 合盖长期运行可用
  `pnpm start:awake`（内部执行 `caffeinate -i pnpm start`）。代码修改后必须先构建
  并重启，不能只改 `src/`。
- 使用 `pnpm status`、`pnpm stop`、`pnpm restart` 管理 Node 机器人；`pnpm stop` 只停止 AI 回复，不停止普通 QQ 或 NapCat 小号。
- 重启会清空内存中的对话历史，这是当前设计，不是数据丢失故障。

## 代码导航

- `src/config.ts`：环境变量校验、白名单、Codex 与主动互动参数。
- `src/persona.ts`：铃铃酱完整内置默认提示词，必须与环境模板和人设文档一致。
- `src/create-bot.ts`：OneBot 事件入口、限流、队列、图片加载和回复装配。
- `src/engagement-state.ts`：主动文字、轻量表情、固定任务和热点调度的最小持久状态。
- `src/daily-image-archive.ts`：投稿原图的日期目录、连续编号与本机持久化。
- `src/daily-longevity.ts`：21:50 私聊征集、归档、内存待发送队列、22:00 逐图审核与群发编排。
- `src/group-chat-handler.ts`：命令、会话上下文和 AI 对话编排。
- `src/group-participation.ts`：群级短期上下文、话题参与、定时救场、早报、批斗、
  热点、旧梗和轻量表情编排。
- `src/onebot/message.ts`：私聊、群聊、`@`、CQ 码和图片消息解析。
- `src/onebot/image-loader.ts`：QQ 图片读取、大小/格式/来源限制和 data URL 转换。
- `src/ai/codex-cli-ai.ts`：受限 Codex 非交互调用、JSONL 结果解析和生成图片读取。
- `scripts/macos/`：普通 QQ 与 NapCat 小号双启动及恢复保护。

遇到运行问题先复现和读取日志，再按 `docs/OPERATIONS.md` 排查，不要先大范围改代码。
